Dans cette section nous allons vous dévoiler, comment construire un prompt parfait pour faire évoluer vos projets Singulary.


<Step>
<StepItem title="Comprendre votre besoin">

Avant de commencer à poser une question a Singulary vous devez définir votre besoin, **Mon besoin concerne-t-il le backend, le frontend ?, une api ? ...**.

En fessant cela, vous resserez le champ des possibles et vous multipliez par 10 vos chances d'obtenir une réponse pertinante.


</StepItem>
<StepItem title="Constuire votre question">
Un des gros points fort de Singulary est que vous pouvez générer automatiquement des documentations pour vos projets. Imaginons que vous souhaitez partager a vos collaborateurs votre vision, vous pouvez.

Pour ce faire vous n'avez qu'a créer un projet Singulary, vous rendre ensuite dans l'outil **Documentation** et cliquer sur générer.

</StepItem>

</StepItem>
</Step>
